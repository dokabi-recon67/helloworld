How It Works

1

TLS Wrapper

All traffic wrapped in TLS on port 443. Indistinguishable from normal HTTPS.

2

SSH Transport

Battle-tested SSH encryption inside. No custom crypto. No vulnerabilities.

3

Full Tunnel

Route all system traffic through your server. Your IP becomes the VM's IP.

4

Tor Mode

Optional Tor exit with circuit rotation. Change your IP on demand.

Quick Start

1. Set Up Your VM

wget https://helloworld.example/install.sh
chmod +x install.sh
sudo ./install.sh
sudo helloworld-server setup
sudo helloworld-server start

2. Add Your SSH Key

# On your VM
echo "your-ssh-public-key" >> ~/.ssh/authorized_keys

3. Configure Client

Open HelloWorld, add your server (hostname, key path), and click Connect.

4. Verify

Visit api.ipify.org - you should see your VM's IP.

Requirements

Client

  • Windows 10+ or macOS 11+
  • OpenSSH (included in OS)

Server

  • Linux VM (any cloud provider)

  • Root access
