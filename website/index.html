How It Works

1

TLS Wrapper

All traffic wrapped in TLS on port 443. Indistinguishable from normal HTTPS.

2

SSH Transport

Battle-tested SSH encryption inside. No custom crypto. No vulnerabilities.

3

Full Tunnel

Route all system traffic through your server. Your IP becomes the VM's IP.

4

Tor Mode

Optional Tor exit with circuit rotation. Change your IP on demand.

Quick Start Guide

Choose your cloud provider: Oracle Cloud Guide | Google Cloud Guide

1

Create Oracle Cloud Account

Go to cloud.oracle.com and click Sign Up.

  • Enter email, verify it
  • Fill in name, address, phone
  • Add credit card (for verification only - you will NOT be charged)
  • Wait for account activation
2

Create a Network (VCN)

Menu (☰) > Networking > Virtual Cloud Networks > Start VCN Wizard

  • Select "Create VCN with Internet Connectivity"
  • VCN Name: helloworld-vcn
  • IPv4 CIDR: 10.0.0.0/16 (keep default)
  • DNS hostnames: Check the box
  • IPv6: Click "Add IPv6 Prefix" > "Oracle allocated /56"
  • Click Next, then Create
3

Open Port 443 (Security Rules)

In your VCN: Public Subnet > Default Security List > Add Ingress Rules

  • Rule 1: Source 0.0.0.0/0, TCP, Port 443
  • Rule 2: Source ::/0, TCP, Port 443
  • Click Add Ingress Rules

These rules allow your tunnel traffic through the firewall

4

Create Your VM Instance

Menu (☰) > Compute > Instances > Create Instance

Image & Shape:

  • Image: Oracle Linux 9
  • Shape: VM.Standard.A1.Flex (Always Free!)
  • OCPUs: 2, Memory: 12 GB

Networking:

  • VCN: Select helloworld-vcn
  • Subnet: Public Subnet (or create new public subnet)
  • Public IPv4: CHECK THIS BOX!

SSH Keys:

  • Select "Generate a key pair for me"
  • Click Download private key - SAVE THIS FILE!
  • Click Download public key - Save this too

Auto-Setup (Recommended!)

In "Management" section, upload this file to auto-install everything:

Download cloud-init.yaml

Show advanced options > Management > Choose cloud-init script file

Click Create and wait 2-5 minutes for "Running" status

5

Get Your Server IP

Once your instance shows "Running":

  • Find "Public IP address" on the instance page
  • Copy this IP (e.g., 129.153.xxx.xxx)
  • Save it - you'll need it for the client app!

If you used cloud-init, wait 5 more minutes for auto-setup to complete

6

Manual Install (Skip if you used cloud-init)

Connect to your VM via SSH and run:

curl -sSL https://raw.githubusercontent.com/dokabi-recon67/helloworld/main/scripts/install_server.sh | sudo bash

SSH username is opc for Oracle Linux

7

Download Client and Connect

  • Download HelloWorld for Windows/Mac
  • Extract ZIP and run helloworld.exe
  • Click + ADD button
  • Enter: Name (anything), Host (your VM's IP), Port (443)
  • Browse to your SSH private key file (.key)
  • Username: opc
  • Click Add Server, then CONNECT
8

Verify It Works!

Open your browser and go to: api.ipify.org

  • You should see your VM's IP, not your home IP
  • If not, enable Full Tunnel Mode in the app

Congratulations! Your traffic is now routed through your server!

Get Your Server IP

4

Set Up SSH Keys

On your PC (PowerShell):

cd $env:USERPROFILE\.ssh
ssh-keygen -t ed25519 -f helloworld_key -N '""'
Get-Content helloworld_key.pub

Add to Google Cloud VM:

  • Click "SSH" button on your VM (opens browser terminal)
  • Run: mkdir -p ~/.ssh && nano ~/.ssh/authorized_keys
  • Paste your public key, save (Ctrl+X, Y, Enter)
  • Run: chmod 600 ~/.ssh/authorized_keys
5

Wait for Auto-Setup

If you used the startup script, wait 5 minutes, then verify:

7

9

Verify It Works!

  • If not, enable Full Tunnel Mode in the app

Congratulations! Your traffic is now routed through your server!

Advanced: Manual Server Setup (Terminal)

If you prefer to set up manually:

# Download server package
wget https://github.com/dokabi-recon67/helloworld/releases/latest/download/helloworld-server.tar.gz
tar xzf helloworld-server.tar.gz
cd helloworld-server

# Build from source
mkdir build && cd build
cmake ..
make
sudo make install

# Run setup wizard
sudo helloworld-setup

# Or configure manually
sudo helloworld-server setup
sudo helloworld-server start

Server Commands

# Check status
sudo helloworld-server status

# Enable Tor mode (changes your exit IP)
sudo helloworld-server tor

# Get a new Tor IP
sudo helloworld-server rotate

# Switch back to direct mode
sudo helloworld-server direct

# Show current public IP
sudo helloworld-server ip
