How It Works

1

TLS Wrapper

All traffic wrapped in TLS on port 443. Indistinguishable from normal HTTPS.

2

SSH Transport

Battle-tested SSH encryption inside. No custom crypto. No vulnerabilities.

3

Full Tunnel

Route all system traffic through your server. Your IP becomes the VM's IP.

4

Tor Mode

Optional Tor exit with circuit rotation. Change your IP on demand.

Quick Start Guide

Need detailed step-by-step with screenshots? See the Complete Oracle Cloud Setup Guide

1

Create Oracle Cloud Account

Go to cloud.oracle.com and sign up for free. Add a card for verification (you will NOT be charged for Always Free resources).

2

Create a Network (VCN)

In Oracle Cloud Console: Networking > Virtual Cloud Networks > Start VCN Wizard > Select "Create VCN with Internet Connectivity".

  • VCN Name: helloworld-vcn
  • IPv4 CIDR Block: 10.0.0.0/16 (default)
  • Use DNS hostnames: Checked
  • Add IPv6 Prefix: Click Add IPv6 Prefix > Select Oracle allocated /56
3

Open Port 443

In your VCN, click Public Subnet > Default Security List > Add Ingress Rules:

  • Source CIDR: 0.0.0.0/0 (IPv4)
  • Source CIDR: ::/0 (IPv6)
  • Protocol: TCP, Port: 443

Add both rules - one for IPv4, one for IPv6!

4

Create Your VM

Compute > Instances > Create Instance

  • Image: Oracle Linux 9 or Ubuntu
  • Shape: VM.Standard.A1.Flex (Always Free!)
  • OCPUs: 2-4, Memory: 12-24 GB (all free)
  • Network: Your VCN, Public Subnet
  • Public IP: Assign public IPv4
  • SSH Key: Generate key pair and SAVE both files!
5

Install HelloWorld Server

Connect to your VM via SSH and run:

curl -sSL https://raw.githubusercontent.com/dokabi-recon67/helloworld/main/scripts/install_server.sh | sudo bash

Username is opc for Oracle Linux, ubuntu for Ubuntu

6

Download Client and Connect

  • Download HelloWorld for Windows/Mac
  • Run the app, click + ADD SERVER
  • Enter your VM's IP, port 443, browse to your SSH key
  • Click CONNECT

Verify at api.ipify.org - shows your VM's IP!

Advanced: Manual Server Setup (Terminal)

If you prefer to set up manually:

# Download server package
wget https://github.com/dokabi-recon67/helloworld/releases/latest/download/helloworld-server.tar.gz
tar xzf helloworld-server.tar.gz
cd helloworld-server

# Build from source
mkdir build && cd build
cmake ..
make
sudo make install

# Run setup wizard
sudo helloworld-setup

# Or configure manually
sudo helloworld-server setup
sudo helloworld-server start

Server Commands

# Check status
sudo helloworld-server status

# Enable Tor mode (changes your exit IP)
sudo helloworld-server tor

# Get a new Tor IP
sudo helloworld-server rotate

# Switch back to direct mode
sudo helloworld-server direct

# Show current public IP
sudo helloworld-server ip
