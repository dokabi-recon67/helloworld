How It Works

1

TLS Wrapper

All traffic wrapped in TLS on port 443. Indistinguishable from normal HTTPS.

2

SSH Transport

Battle-tested SSH encryption inside. No custom crypto. No vulnerabilities.

3

Full Tunnel

Route all system traffic through your server. Your IP becomes the VM's IP.

4

Tor Mode

Optional Tor exit with circuit rotation. Change your IP on demand.

Quick Start Guide

1

Create a Free Cloud VM

Sign up for Oracle Cloud Free Tier and create a small Linux VM (Ubuntu recommended).

Make sure to save the SSH key file they give you!

2

Open Port 443 on Your VM

In your cloud dashboard, find "Security Rules" or "Firewall" and add a rule to allow incoming traffic on port 443 (TCP).

3

Set Up the Server

Connect to your VM and run the automatic installer:

  • Open your VM's terminal (via SSH or web console)
  • Copy and paste this one command:
curl -sSL https://raw.githubusercontent.com/dokabi-recon67/helloworld/main/scripts/install_server.sh | sudo bash

This installs everything automatically!

4

Download the Client App

Download HelloWorld for your computer:

  • Windows: HelloWorld-win64.zip
  • macOS: HelloWorld-macos.zip

Extract the ZIP and run helloworld.exe (Windows) or HelloWorld.app (Mac).

5

Add Your Server

In the HelloWorld app, click Settings and edit the config file:

  • name = Any name (e.g., "My VPS")
  • host = Your VM's IP address
  • port = 443
  • key = Path to your SSH key file
  • user = root (or ubuntu)
6

Connect!

Select your server from the dropdown and click CONNECT.

To verify it's working, visit api.ipify.org - you should see your VM's IP address!

Advanced: Manual Server Setup (Terminal)

If you prefer to set up manually:

# Download server package
wget https://github.com/dokabi-recon67/helloworld/releases/latest/download/helloworld-server.tar.gz
tar xzf helloworld-server.tar.gz
cd helloworld-server

# Build from source
mkdir build && cd build
cmake ..
make
sudo make install

# Run setup wizard
sudo helloworld-setup

# Or configure manually
sudo helloworld-server setup
sudo helloworld-server start

Server Commands

# Check status
sudo helloworld-server status

# Enable Tor mode (changes your exit IP)
sudo helloworld-server tor

# Get a new Tor IP
sudo helloworld-server rotate

# Switch back to direct mode
sudo helloworld-server direct

# Show current public IP
sudo helloworld-server ip

Requirements

Server

  • Port 443 open
  • Root/sudo access

Recommended
